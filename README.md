## コンパイル・実行方法
`sample.c`/`sample.java`をコンパイル・実行する場合の例．

### `.c`ファイル
`gcc`コマンドにファイル名を指定してコンパイル
```bash
$ gcc sample.c
```
正常にコンパイルできれば，ソースコードのファイル名にかかわらず`a.out`が生成される．
```bash
$ ./a.out
```

### `.java`ファイル
`javac`コマンドにファイル名を指定してコンパイル
```bash
$ javac Sample.java
```
正常にコンパイルできれば，`Sample.class`が生成される．
```bash
$ java Sample
```
